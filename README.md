# Passive Microfluidic Micromixer Design

## Overview
Capstone project focused on improving mixing efficiency in
laminar microfluidic flows for lab-on-a-chip applications.

## Objectives
- Enhance mixing without external energy input
- Analyze flow, pressure, and diffusion behavior

## Tools Used
- SolidWorks
- COMSOL Multiphysics

## Methodology
- Designed multiple 2D channel geometries
- Simulated laminar flow and species transport
- Evaluated mixing index performance

## Results
- Achieved ~35% improvement in mixing efficiency
- Identified optimal channel geometry

## What I Learned
- Microfluidic flow behavior
- CFD simulation techniques
- Engineering trade-off analysis
